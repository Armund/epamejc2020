Задача: Написать программу которая генерирует новую строку на основании входных данных, строка + четные/нечетные.
Функционал программы:
1. Программа предлагает пользователю ввести строку для обработки.
2. В случае корректного ввода строки пользователю предлагается ввод типа индекса, символы с которым
будут удалены из введенной строки.
3. В случае корректного ввода программа генерирует новую строку на основе указанного фильтра, выводит ее в консоль и
предлагает пользователю повторить процедуры 1-3.
3.1. Для выхода из программы в п.1 или п.2 следует ввести строку "exit".
3.2. В случае некорректного воода строки (длина == 0) или типа индекса, программа будет предлагать
пользователю повторить ввод.
