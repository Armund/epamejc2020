Задача: Посчитать количество чисел в строке.
Функционал программы:
1. Программа предлагает пользователю ввести строку с длиной > 0.
2. В случае успешного ввода строки программа рекурсивно вычисляет количество числовых символов в строке
и выводит результат в консоль.
3. Программа предлагает ввести новую строку (п.1). Для выхода из программы необходимо ввести -1.
