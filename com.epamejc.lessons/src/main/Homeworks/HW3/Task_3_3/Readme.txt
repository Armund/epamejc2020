Задача: Поменять местами символы в строке, на вход приходит строка и 2 индекса между, которыми нужно сделать обмен.
Функционал программы:
1. Программа предлагает пользователю ввести строку для обработки.
2. В случае корректного ввода строки пользователю предлагается ввод двух индексов
для замены символов во введенной строке. 2 целочисленных индекса должны быть разделены любым сепаратором,
не являющимся буквенным символом. Допускается любое количество пробелов между индексами и разделителем.
3. В случае корректного ввода строки и индексов программа заменяет символы во введенной строке и выводит ее в консоль.
4. Пользователю предлагается ввод новых индексов для замены (п.2). Для выхода из программы необходимо ввести -1
во время ввода индексов.
