Задача: Сгенерировать массив массивов указав размерность, заполнить его случайными символами.
Написать 2 стратегии, которые будут формировать строку на основе данного массива.
Стратегия А - работает по принципу *_*_*, а стратегия B - работает по принципу _*_*_ .
Функционал программы:
1. Пользователю предлагаются условия ввода:
1.1. Количество столбцов и ячеек должно быть целым положительным числом в промежутке от 1 до 85.
1.2. Для выхода из программы нужно ввести -1 (либо при указании количества строк, либо при указании кол-ва столбцов).
2. В случае корректного ввода программа генерирует массив в заданном диапазоне и заполняет его случайными символами.
3. Пользователю предлагается ввести номер стратегии (1 - четные индексы, 2 - нечетные).
4. В случае корректного ввода программа печатает сгенерированный массив в консоль, номер стратегии,
а затем строку, сгенерированную на основании массива и стратегии.
5. В случае некорректного ввода либо количества строк, либо количества столбцов, либо номера стратегии,
пользователю выводится сообщение об ошибке и предлагается повторный ввод некорректного значения.
6. На любой стадии ввода данных (п.2 , п.3) возможно завершение программы с помощью ввода значения -1.
