Задача: Поиск числа Фиббоначи.
Функционал программы:
1. Программа предлагает пользователю ввести целочисленную переменную для расчета
числа Фибоначчи в диапазоне 1 - 30.
2. В случае успешного ввода исходного значения программа вычисляет число Фибоначчи с помощью рекурсивной функции
и выводит его в консоль.
3. Программа предлагает ввести новую целочисленную переменную для расчета (п.1). Для выхода из программы
надо ввести в консоль -1.
