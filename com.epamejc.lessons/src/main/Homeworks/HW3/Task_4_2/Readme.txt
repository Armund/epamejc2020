Задача: Написать программу которая умеет вовзодить в степени.
Функционал программы:
1. Программа предлагает пользователю ввести целочисленное значение для базы в диапазоне 1 - 15.
2. В случае успешного ввода базы программа предлагает ввести целочисленное значение для степени
в диапазоне 1 - 15.
3. В случае успешного ввода базы и степени программа вычисляет конечное значение с помощью рекурсивной функции и
выводит значение в консоль.
4. Программа предлагает ввести новые значения базы (п.1) и степени (п.2). Для выхода из программы
необходимо ввести -1 при вводе базы или степени.
