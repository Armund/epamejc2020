Задача: Создать массив массивов: задается размерность, на основании которой генерируются случайные символы типа char.
Функционал программы:
1. Пользователю предлагаются условия ввода:
1.1. Количество столбцов и ячеек должно быть целым положительным числом в промежутке от 1 до 85.
1.2. Для выхода из программы нужно ввести -1 (либо при указании количества строк, либо при указании кол-ва столбцов).
2. В случае корректного ввода программа генерирует массив в заданном диапазоне, заполняет его случайными символами и
печатает в консоль.
3. В случае некорректного ввода либо количества строк, либо количества столбцов, пользователю выводится сообщение
об ошибке и предлагается повторный ввод некорректного значения.
