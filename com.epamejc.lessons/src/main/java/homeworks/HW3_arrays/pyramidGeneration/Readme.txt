Задача: Написать программу которая будет генерировать пирамидку.
Функционал программы:
1. Программа предлагает ввести высоту пирамиды (равнобедренный треугольник) от 1 до 55.
2. В случае корректного ввода программа печатает в консоль пирамиду с заданными параметрами.
2.1. Для выхода из программы надо ввести -1.
