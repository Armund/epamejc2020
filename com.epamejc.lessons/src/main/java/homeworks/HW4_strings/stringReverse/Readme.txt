Задача: Написать программу которая переворачивает слова, но при этом оставляет их на прежних местах.
Функционал программы:
1. Программа предлагает пользователю ввести несколько строк для обработки, разделенных пробелом.
2. В случае корректного ввода строки программа осуществляет реверс строк, причем сохраняя изначальный порядок, а далее
выводит оригинальную и измененную строки в консоль. Триггером для отделения строк друг от друга является пробел.
3. Программа предлагает повторить п.1, для выхода из программы надо ввести -1.
