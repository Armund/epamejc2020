Класс Helper помогает при вводе данных юзером в консоль:
1) Обрабатывает ошибки ввода
2) Обрабатывает ошибки при парсинге (из строки в число, просит ввести повторно)
3) При вводе числа убирает все пробелы вокруг и между цифрами
4) При вводе пустой строки или exit возвращает строку Exit, для выхода из приложения.
Если подразумевается другой способ закрытия приложения, нужно вызвать метод closeReader()
