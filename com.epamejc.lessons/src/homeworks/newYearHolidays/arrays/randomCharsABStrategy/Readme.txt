Задача: Сгенерировать массив массивов указав размерность, заполнить его случайными символами. Написать 2 стратегии, которые будут формировать строку на основе данного массива. Стратегия А - работает по принципу *_*_*, а стратегия B - работает по принципу _*_*_ . Пример:
        input: 5 5
        output:
        d a l p j
        v c b z g
        a d l r f
        f d m x h
        k d l h b
        input strategy: A
        output: dljalfklb
