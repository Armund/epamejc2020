Задача: Поиск числа Фиббоначи
Input: 10
Output: 55

Решение:
Программа просит ввести порядковый номер числа Фибоначчи, и выводит значение этого числа.
Программа зациклена.
Также реализован подсчет времени выполнения вычислений (после 30-го числа Фибоначчи производительность быстро падает).
Верхняя граница ввода - 50.
