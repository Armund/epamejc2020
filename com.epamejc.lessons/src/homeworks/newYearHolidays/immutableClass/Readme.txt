Задача: Создать immutable class, создать пару объектов, посмотреть как работает,
поэксперементировать с данными. ЗАПОМНИТЬ!

Решение:
Был создан класс WordNotSparrow (слово не воробей), в котором есть 2 неинициализированных финальных поля -
word и person, а также конструктор. В момент создания нового обьекта финальные поля инициализируются, и больше
возможности их изменить нет. Слово не воробей, вылетит - не изменишь!
